# 番茄时钟

[English](README.md) | [中文](README.zh-CN.md)

这是一个简单的番茄时钟应用，使用HTML和JavaScript实现。

## 功能

- **开始**: 开始计时25分钟。
- **暂停**: 暂停计时。
- **重置**: 重置计时器到25分钟。


## 使用方法

1. 克隆或下载此仓库。
2. 打开 `index.html` 文件在浏览器中查看番茄时钟。

## 代码说明

### `index.html`

这是主HTML文件，包含了番茄时钟的结构和样式。

### `script.js`

这是包含番茄时钟逻辑的JavaScript文件。
